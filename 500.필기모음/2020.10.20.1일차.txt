[ HTML기본 ]

1. HTML
Hyper Text Markup Language
하이퍼텍스트: 순서없는 글자문서
마크업랭귀지: 웹문서구조를 정하는 언어

2. http
Hyper Text Transper Protocol
하이퍼 텍스트 전송규약
- 전송규약 :  웹문서가 전송되기위한 기술적 약속
https -> s는  security 보안접속 웹페이지

3. meta정보란?
- 문서를 위한 필수정보
- 문서제목, 문서요약내용,문서인코딩 정보 등
- meta태그로 표현
  charset 속성은 charactor set 문자인코딩 셋
 -> utf-8 형식은 2byte문자도 안깨지는 형식

4. 내용은 body태그 내부에 기술함

5. 코드 편집기
- 메모장(비추)
-> 브라켓(adobe) : 정식사용툴(무료)
- 서브라임텍스트3, 에디트플러스, 아톰
  비주얼스튜디오 코드


6. 브라켓 확장 플러그인
- Emmet : 코딩을 줄여주는 플러그인
- Beautify : 소스코드 자동줄맞춤 플러그인
- Custom Work : 탭메뉴 플러그인
- Brackets Snippets (by edc) : 소스코드 저장 플러그인

: 확장기능관리자를 클릭하여 검색 후 설치함

7. 태그의 중요개념
- 부모요소: 자식요소를 가진요소
- 자식요소: 자신을 싸고 있는 요소가 있는 요소
- 줄맞춤은 부모 아래 자식요소를 탭(tab)하나만큼 들여서 코딩한다.(가독성때문에)
