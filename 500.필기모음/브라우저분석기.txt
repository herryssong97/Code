[ 브라우저 분석기 - F12 또는 ctrl+shift+I ]
- 크롬브라우저를 기본으로 한다.

1) 요소분석기(element)
- 직접선택툴로 요소를 선택하여 소스를 볼 수 있다.
- 소스를 클릭하여 요소에 표시할 수 있다.
- DOM(Document Object Model)분석을 할 수 있다
 (부모자식 관계 파악)

2) 네트워크 분석기(Network)
- 종류별 데이터의 로딩 속도체크를 할 수 있다
- 인터넷 임시파일에 저장된 파일을 저장할 수 있다
- 이미지만 별도로 선택하여 받아진 이미지를 저장 할 수 있다
-> 이미지저장 불편함!
   -> 다른방법??? 크롬확장프로그램을 설치한다.
   : Image downloader 검색하여 설치
1) 크롬 브라우저 설정 아이콘 클릭
2) 도구더보기 > 확장프로그램 을 클릭
3) 햄버거 버튼 클릭 -> Chrome 웹 스토어 가기!
4) 검색에 "Image downloader"를 검색함
5) 설치후 아이콘을 오른쪽 상단에 고정함
