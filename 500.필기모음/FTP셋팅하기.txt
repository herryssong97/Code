[ FTP 연결하기 ]
1. FTP 프로그램을 설치한다.
(파일질라, 알드라이브 등)
2. 프로그램을 실행하여 연결정보를 넣는다
 1) 호스트: FTP IP 주소
    - 닷홈 호스팅 정보에서 웹서버 아이피 주소를 복사함
 2) 아이디(사용자명)
    - 닷홈 호스팅 정보에서 FTP 아이디를 복사함
 3) 비밀번호
    - 닷홈 호스팅시에 지정한 비밀번호를 넣는다
    (만약 안되면 비빌번호 수정으로 다시 설정후 접속한다)
 4) 포트번호(21번)

3. 연결후에는 (닷홈기준) html파일 아래에 
   올리고자 하는 파일을 드래그에 드롭한다!

4. 업로드 후엔 접속주소를 써서 브라우저에서 열어본다
http://닷홈호스팅아이디.dothome.co.kr/올린폴더명/올린파일명.html

  http://닷홈호스팅아이디.dothome.co.kr/
 -> 여기까지가 내가올린 서버의 Root(html폴더)가 된다!

예)
http://tombap8.dothome.co.kr/02.Drama_PJ/Dokkaebi.html

   ※ 참고로 닷홈 서버는 대소문자를 철저히 구분한다!!!


5. 루트폴더 지정하기
  1) 파일질라
   - 파일 > 사이트관리자 : 자기사이트 지정하기!
   - 자기사이트 저정된 옵션 상단 탭에서 "고급"에서
	"기본 리모트 디렉터리"에 루트폴더명을 써준다!
	이렇게! -> /html
  2) 알드라이브
   - 사이트맵 창에서 지정한 자기사이트 설정 정보 하단에
	"서버기본경로" 에 "/html"을 써준다!

6. 자기 포폴 사이트 폴더를 업로드 후 주소를 정확히 알아낸다!

예)
http://tombap8.dothome.co.kr/movie_PJ/

7. 만약 루트 주소만 쓰고 아무 파일명을 안쓰면 
   루트에 있는 index파일명을 가진 html을 열어준다.
   (또는 하위폴더명까지만 쓰면 자동으로 아래 있는 
    index파일을 열어준다)
