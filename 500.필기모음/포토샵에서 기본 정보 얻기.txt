[ 포토샵에서 기본 정보 얻기 ]
1. 가로, 세로크기 : 사각 선택툴
  - 드래그 상태에서 가로,세로 크기 자동표시
  - 드래그 상태에서 스페이스바 누르고 위치이동
  - 드래그 상태에서 shift를 누르면 정사각형유지
  - 드래그가 끝나면 크기정보는 info(F8)에서 확인!

2. 위치,여백 : 룰러(ruler)
  - 상단영역의 정보툴 중 ruler를 선택
  - 자모양의 툴로 클릭 드래하여 거리를 잰다
    ( 혹시 모양이 안나오면 CapsLock키가 켜진것임)
  - 정 각도 방향으로 재려면 Shift를 누르고 드래그함
  - 측정된 정보는 info패널에 표시됨

3. 색상정보: Eyedropper 툴(스포이드모양)
  - 원하는 색상위에서 클릭하면 전경색에 색상이 들어감
  - 툴박스의 전경색 부분을 클릭하여 color picker 패널을
    띄워 놓고 색상을 계속 선택하면 선택된 색상정보가
    컬러픽커 패널에 표시된다(해시코드, rgb코드, hsl코드)

4. 글자정보: character 패널에 정보를 얻는다
- 글자 레이어가 별도로 존재한다. 이 레이어를 선택함
- 글자체, 글자 크기, 글자 자간, 글자 두께 등
- window > character 패널을 선택하여 열기
- 주의: 72ppi 해상도인 포토샵에서는 px과 pt단위 같다
그러나 브라우저에서는 px과 pt가 다르다!
(예: 브라우저에서 12pt = 16px 같은 크기임)
포토샵에서는 pt단위이라도 px로 읽어도됨!

5. 정보표시 패널 : info(F8)

- 표시단위를 px로 변경함
: ctrl+K 환경설정파일에서 
Unit & Rulers (단위)에서 변경!
-> Units : Rulers에서 px단위설정
(Edit>Preferences)
